【算法复盘】从一维到高维：利用单调队列进行降维搜索

Posted on 2025-12-29 Edited on 2025-12-30 In CS Study

Problem Statement:
P2216 理想的正方形

  • 来源: HAOI 2007 / Luogu
  • Algorithm: Monotonic Queue, 2D RMQ
  • Difficulty: Green (普及+/提高)

在解决区间最值查询（Range Minimum/Maximum Query, RMQ）问题时，线段树和 ST 表常被视为标准解法。然而，当问题场景从静态转为动态滑动窗口，甚至扩展至二维矩阵时，单调队列 (Monotonic Queue) 凭借其 O(N)O(N)O(N) 的线性复杂度，展现出了优于 O(NlogN)O(N \log N)O(NlogN) 算法的工程美学。

本文以 P2216 [HAOI2007] 理想的正方形 为例，复盘我如何利用“降维思想”将二维 RMQ 问题拆解为两次一维扫描。

1. 问题分析 (Problem Analysing)

给定一个 a×ba \times ba×b 的整数矩阵，我们需要在一个 n×nn \times nn×n 的正方形区域内，找到该区域所有元素的最大值与最小值的差值，并使这个差值最小。

数学语言描述如下：
定义区域 S(i,j)S(i, j)S(i,j) 为以 (i,j)(i, j)(i,j) 为右下角的 n×nn \times nn×n 子矩阵。
我们需要求：

min_n≤i≤a,n≤j≤b(max_(x,y)∈S(i,j){Val(x,y)}−min_(x,y)∈S(i,j){Val(x,y)})\min\_{n \le i \le a, n \le j \le b} \left( \max\_{(x,y) \in S(i,j)} \{Val(x,y)\} - \min\_{(x,y) \in S(i,j)} \{Val(x,y)\} \right) min_n≤i≤a,n≤j≤b(max_(x,y)∈S(i,j){Val(x,y)}−min_(x,y)∈S(i,j){Val(x,y)})

数据范围：a,b≤1000a, b \le 1000a,b≤1000。
如果采用暴力枚举每个正方形再遍历求最值，复杂度为 O(a · b · n2)，这将达到 109 数量级，必然 TLE。我们需要一个 O(a⋅b)O(a \cdot b)O(a⋅b) 的解法。

2. 核心思想：算两次降维

解决高维问题的通用策略是：分治与降维。
二维滑动窗口本质上是两个维度上的独立滑动。我们可以将其拆解为：先处理行，再处理列。

Step 1: 行压缩

首先，我们将每一行视为一个独立的一维数组。
利用单调队列，我们可以计算出每一行中，以 jjj 为结尾的长度为 nnn 的窗口最值。

定义辅助矩阵 RowMax[i][j]RowMax[i][j]RowMax[i][j]：

RowMax[i][j]=max{Val(i,k)∣j−n+1≤k≤j}RowMax[i][j] = \max \{ Val(i, k) \mid j-n+1 \le k \le j \} RowMax[i][j]=max{Val(i,k)∣j−n+1≤k≤j}

这步操作将 NNN 个横向的格子，压缩成了一个代表“横向最值”的点。

Step 2: 列压缩

接下来，我们基于 RowMaxRowMaxRowMax 矩阵进行纵向扫描。
对于每一列 jjj，我们在 RowMaxRowMaxRowMax 上跑单调队列。

定义最终状态 FinalMax[i][j]FinalMax[i][j]FinalMax[i][j]：

FinalMax[i][j]=max{RowMax(k,j)∣i−n+1≤k≤i}FinalMax[i][j] = \max \{ RowMax(k, j) \mid i-n+1 \le k \le i \} FinalMax[i][j]=max{RowMax(k,j)∣i−n+1≤k≤i}

具体含义：FinalMax[i][j]FinalMax[i][j]FinalMax[i][j] 表示的是第 jjj 列中，连续 nnn 个“横向最大值”中的最大值。
不难得出，这等价于整个 n×nn \times nn×n 区域的最大值。

通过这两次扫描，我们将二维区域最值问题，转化为了 2×a2 \times a2×a 次一维单调队列操作。

3. 算法实现与复杂度 (Implementation)

单调队列的核心在于维护一个双端队列 (Deque)，存储元素的下标。（学习该板块时常会无端想起双指针）

  • 入队时：维护单调性。若队尾元素劣于当前元素（比当前小且更早出队），则淘汰队尾。
  • 出队时：维护时效性。若队头下标滑出窗口范围，则淘汰队头。

以下是 C++ 核心代码片段，演示了单调队列的基本实现：

// 通用单调队列处理函数
// arr: 输入数组, res: 输出结果, type: 0求min, 1求max
void get_window_val(int *arr, int *res, int len, int k, int type) {
    deque<int> q;
    for (int i = 1; i <= len; i++) {
        // 1. 维护单调性 (Eliminate inferior elements)
        if (type == 0) { // Min
            while (!q.empty() && arr[q.back()] >= arr[i]) q.pop_back();
        } else {         // Max
            while (!q.empty() && arr[q.back()] <= arr[i]) q.pop_back();
        }
        
        q.push_back(i); // Push index
        
        // 2. 维护时效性 (Remove outdated elements)
        if (q.front() <= i - k) q.pop_front();
        
        // 3. 记录结果
        if (i >= k) res[i] = arr[q.front()];
    }
}
